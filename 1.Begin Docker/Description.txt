Докер робити ізольоване середовище і робити свою ОС.
Докер коли у ньому розгртаєтсья сайт він працює на своїй операційній системі.
Докер - може бути windows, але це не добре, бо контейнери по різному працюють.
Тому усі докери і усі контейнери працюють в Linux.
Linux - версії є наступні - Ubunty, Debian, MacOS.
Тобто, щоб нам працювати із Docker - ми маємо мати Linux.

Docker - програма, яка ставиться в Linux.

WSL - це вбудована віртуальна машина для Windows для запуску Linux (Debian, Ubuntu)
Для роботи з Linux у віндовс Вам нічого не потрібно, раніше ставили - Virtual Box.
